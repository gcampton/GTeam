## Search Query Deep-Dive

**Use when:** Optimising an established PPC account with 3+ months of data. Goes deeper than the standard audit.

**Query-level analysis workflow:**

1. **Export search term report** — 90 days minimum, include: search term, campaign, ad group, impressions, clicks, cost, conversions, conversion value

2. **Classify queries by intent:**

| Intent | Signal | Action |
|---|---|---|
| High commercial | "buy", "price", "best", "vs", "review" | Bid aggressively, dedicate budget |
| Informational | "how to", "what is", "guide" | Lower bids or exclude (unless content funnel) |
| Navigational | Brand names, competitor names | Separate campaigns, different bidding |
| Transactional | "discount", "coupon", "free trial" | High intent — ensure landing page matches |

3. **Negative keyword mining:**
   - Sort by cost descending, filter zero conversions
   - Group waste into themes (wrong industry, wrong product, wrong intent)
   - Add as phrase or exact negatives at appropriate level (account/campaign/ad group)
   - Calculate total waste recovered as % of spend

4. **Query-to-keyword gap analysis:**
   - High-converting search terms NOT in keyword list → add as exact/phrase
   - Keywords with zero impressions → check match type, bid, quality score
   - Queries triggering wrong ad group → restructure or add negatives

5. **ROAS by query cluster:**
   Group queries into thematic clusters, calculate ROAS per cluster. Shift budget toward highest-ROAS clusters.

**Output:** Query analysis report with waste identified, negatives to add, keywords to add, and budget reallocation recommendations.

# Supermetrics

> Supermetrics is a marketing intelligence platform that connects marketing, ecommerce, and sales data sources to business intelligence tools, data warehouses, cloud storage, and spreadsheets. Used by 200K+ companies in 120 countries to automate data transfers, reduce reporting time, and maintain SOC2/GDPR/CCPA compliance.

## Developer Tools
- [Supermetrics API](https://supermetrics.com/products/api): Supermetrics provides a RESTful API that allows developers to programmatically extract marketing data from 70+ sources and integrate it into custom applications. Returns data in JSON/CSV formats with query management capabilities

## Business Intelligence & Reporting Tools
- [Excel Integration](https://supermetrics.com/products/excel): Supermetrics add-in connects directly to Excel, automatically pulling marketing data from platforms like Google Ads and Facebook into spreadsheet cells. Features scheduled refreshes and pre-built templates for common marketing reports
- [Google Sheets Integration](https://supermetrics.com/products/google-sheets): Supermetrics add-on installs in Google Sheets, enabling users to query marketing platforms and populate sheets with campaign data, metrics, and dimensions automatically with email alerts for data changes
- [Power BI Integration](https://supermetrics.com/products/power-bi): Supermetrics connector appears as a data source in Power BI, allowing users to import marketing data from multiple ad platforms into Power BI datasets and dashboards with drag-and-drop functionality
- [Looker Studio Integration](https://supermetrics.com/products/looker-studio): Supermetrics connector integrates with Looker Studio as a data source, enabling users to create marketing dashboards by connecting advertising platforms directly to Google's visualization tool

## Data Warehouses Connections
- [BigQuery Integration](https://supermetrics.com/products/bigquery): Supermetrics automatically transfers marketing data from advertising platforms into BigQuery tables with customizable schemas, scheduled data loads, and historical data backfills for data warehouse analytics
- [Snowflake Integration](https://supermetrics.com/products/snowflake): Supermetrics pipelines marketing data directly into Snowflake tables, handling data transformations and providing scheduled transfers from advertising platforms to the data warehouse
- [Amazon Redshift Integration](https://supermetrics.com/products/amazon-redshift): Supermetrics loads marketing data from advertising platforms into Redshift clusters, managing table creation, data types, and automated refresh schedules
- [Azure Synapse Integration](https://supermetrics.com/products/azure-synapse): Supermetrics transfers marketing data into Azure Synapse Analytics tables with configurable refresh schedules and automatic data type mapping
- [Google AlloyDB Integration](https://supermetrics.com/products/google-alloydb): Supermetrics moves marketing data from advertising platforms into AlloyDB PostgreSQL tables for integration with application databases
- [Azure SQL Database Integration](https://supermetrics.com/products/azure-sql-database): Supermetrics populates Azure SQL Database tables with marketing data from advertising platforms, enabling SQL queries on marketing performance data

## Cloud Storage Connections
- [Amazon S3 Integration](https://supermetrics.com/products/amazon-s3): Supermetrics exports marketing data from advertising platforms and stores it as files in S3 buckets in various formats (JSON, CSV, Parquet) for data lake storage and processing
- [Google Cloud Storage Integration](https://supermetrics.com/products/google-cloud-storage): Supermetrics delivers marketing data from advertising platforms to Google Cloud Storage buckets, creating structured files that can be processed by other Google Cloud services
- [Azure Storage Integration](https://supermetrics.com/products/azure-storage): Supermetrics uploads marketing data from advertising platforms to Azure Storage containers as structured files for further processing by Azure analytics services

## Data Transfer
- [SFTP Integration](https://supermetrics.com/products/sftp): Supermetrics securely transfers marketing data files from advertising platforms to SFTP servers, delivering scheduled reports and data exports to on-premise systems

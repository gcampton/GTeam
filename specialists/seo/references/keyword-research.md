# Keyword Research & Strategy

> **Confidence:** All recommendations are `[HYPOTHESIS]` (untested best-practice) unless marked `[TESTED: date]` or `[REVISED: date]`. Check `../results/` for empirical outcomes before applying advice.

## Search Intent Classification

Always classify intent before targeting a keyword. Matching content type to intent is more important than keyword density.

| Intent | What user wants | Content type |
|--------|----------------|--------------|
| **Informational** | Learn something | Blog post, guide, tutorial |
| **Commercial investigation** | Compare options before buying | Comparison post, review, best-of list |
| **Transactional** | Buy / sign up / download | Product page, landing page, pricing |
| **Navigational** | Find a specific site/page | Brand terms — optimise but don't target |

## Keyword Research Process

### Step 1: Seed Keywords

Start with 5–10 seed keywords describing the core topic. For each seed:
- Run in Google autocomplete (note suggestions)
- Check "People also ask" and "Related searches" at bottom of SERP
- Use a tool if available: Ahrefs, Semrush, Ubersuggest, or Google Keyword Planner

### Step 2: Expand & Filter

For each seed, collect:
- Monthly search volume (MSV)
- Keyword difficulty (KD) — target KD ≤ 40 for new/low-DA sites, up to 70 for established sites
- Current ranking position (from GSC if available)

**Filter rules:**
- Remove zero-volume keywords (< 10 MSV) unless hyper-targeted commercial intent
- Remove keywords where top 3 results are all major brands (Wikipedia, Amazon, etc.) — unlikely to break in
- Flag any keyword where your site already ranks 4–20 as a **quick win**

### Step 3: Intent Grouping

Group keywords into clusters — one cluster per page, not one keyword per page.

Example cluster for "project management software":
- Primary: "project management software" (transactional, high volume)
- Secondary: "best project management tools", "project management software for small teams"
- LSI: "task tracking", "team collaboration tool", "Gantt chart software"

All of these map to one page, not five.

### Step 4: Quick-Win Identification

Quick wins = keywords where you already rank 4–20 (page 1 bottom or page 2).

These pages need:
1. Title tag / H1 update with exact keyword
2. Content depth improvement (match or beat top 3 competitors)
3. 2–4 new internal links from related pages
4. Schema markup (if missing)

Quick wins typically move within 4–8 weeks with targeted updates.

### Step 5: Competitor Gap Analysis

Find keywords competitors rank for that you don't:
- Method A: Ahrefs → Competitors → Content Gap tool
- Method B: Manual — pick 3 competitors, check their top pages via Ahrefs/Semrush "Top Pages" report
- Method C: Search your primary keywords, note what topics appear in competing results that you haven't covered

For each gap keyword: assign to an existing page or flag for a new page.

## Key Rules

- **One primary keyword per page** — multiple pages targeting the same primary keyword = cannibalisation
- **Keyword in title, H1, first 100 words, at least one H2** — minimum on-page requirements
- **Don't over-optimise** — keyword density above 3% reads as spam; aim for 1–2% with natural variants
- **Track rankings weekly** for target keywords after any change — changes take 2–8 weeks to reflect
